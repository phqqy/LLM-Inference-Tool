## 第 1 天学习笔记（Linux 用户/权限）
### 1. 用户管理核心命令
- useradd 用户名：创建用户
- passwd 用户名：设置用户密码
- usermod -aG 组名 用户名：追加用户到指定组
- userdel -r 用户名：删除用户+家目录

### 2. 文件权限核心命令
- ls -l 文件名：查看文件权限
- chmod 755 文件名：修改权限（所有者rwx，组/其他rx）
- chown 用户：组 文件名： 修改文件所属用户/组
